## 例子

使用 `addEventListener()`:

```js
document.addEventListener('pointerlockchange', (event) => {
  console.log('Pointer lock changed');
});
```

使用 `onpointerlockchange` 事件处理程序属性

```js
document.onpointerlockchange = (event) => {
  console.log('Pointer lock changed');
};
```
